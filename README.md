# Calculator App 🧮

A clean, responsive web calculator with persistent history and keyboard support.

### Features
- **Basic operations**: Add, subtract, multiply, divide
- **Multi-line history**: Shows your last 5 calculations
- **Persistent storage**: History saves between sessions using localStorage
- **Keyboard support**: Type numbers/operators on desktop and hit Enter
- **Clear History button**: Wipe saved calculations with one tap
- **Mobile responsive**: Works perfectly on phones and tablets
- **Installable PWA**: Add to iPhone/Android home screen for offline use

### Tech Stack
- **HTML5** - Structure
- **CSS3** - Styling with Flexbox/Grid
- **Vanilla JavaScript** - Logic, localStorage, keyboard events
- **PWA** - Manifest + Service Worker for installability

### How to Use
1. **Mobile**: Tap the buttons to calculate
2. **Desktop**: Use your keyboard - `123+456` then `Enter`
3. **History**: Last 5 calculations auto-save and appear above the display
4. **Install**: Open in Safari/Chrome → Share → Add to Home Screen

### Setup Locally
1. Clone the repo: `git clone https://github.com/toddhurray/Calculator-app.git`
2. Open `index.html` in your browser
3. No build tools or dependencies needed
